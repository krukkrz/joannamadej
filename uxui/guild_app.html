GUILD APP

Guild Application

The guild app is a social app created to make networking easier. The main feature of the application is the ability to create and join groups. Groups usually focus on business-related topics or users' interests. Within groups, users can create threads, polls, events, or video calls. While working at Monterail software house, I had the opportunity to introduce new features to the Guild app and redesign old ones. Below few of the many tasks that I have done:
  • Creating the possibility to create a conversation dedicated to a specific poll
  • Refreshing process of creating events
  • Creating the possibility to create a conversation dedicated to a specific event
  • Improving the flow of joining the application
